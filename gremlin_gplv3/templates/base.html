GREMLIN

A free, low-code microservices framework designed for NLP, AI and legal applications.

Get Started

Built for Legal Engineers and Data Scientists

GREMLIN's drag-and-drop, web-based, low-code editor lets you quickly write and deploy Python code as robust, enterprise-grade APIs AND user-facing, self-service GUIs. Reuse existing scripts and combine them to create robust, complex behavior without repeating yourself. Thanks to GREMLIN's pure Python core, your services run in a scalable, asynchronous system from day one. Learn more at our GitHub page.

Legal Open Source

Legal Open Source ("Legal OS") is the belief that, just like the law, the software that powers the future of law should be free and open. This means you can use GREMLIN for any project - even commercial projects! Download it, customize it, and future proof your legal workflows!

React

Our React-based frontend incorporates best-in-breed open source projects like React-Diagrams, Semantic-UI, Jsonschema-Forms and more.
